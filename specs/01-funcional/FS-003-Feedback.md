# FS-003 - Feedback

## Objetivo

Mostrar qué cambió después de registrar una acción.

## Debe mostrar

- Acción registrada
- Cambios positivos
- Cambios negativos
- Estado actualizado
- Progreso hacia rasgos
- Rasgos desbloqueados
- Objetos desbloqueados
- Rasgos perdidos si aplica

## Ejemplo

Acción: Leí o estudié

Cambios:

- Intelecto +8
- Energía -2
- Ánimo +2
- Propósito +3

Progreso:

- Lector 2/3
