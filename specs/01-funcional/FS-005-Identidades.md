# FS-005 - Identidades

## Objetivo

Mostrar quién está siendo el personaje.

Las identidades se calculan desde rasgos activos.

## Identidades MVP

- Aprendiz Permanente: requiere Lector.
- Equilibrado: requiere Activo y Exfumador.
- Constructor: requiere Constante.
- Conector Social: requiere Sociable.

## Reglas

- Si no hay rasgos activos, no se muestra identidad dominante.
- Si hay varias identidades, se muestra una principal y el resto como secundarias.
